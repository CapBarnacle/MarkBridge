# Inspection Feature Matrix

## Common Features
- file format
- file size
- page count / sheet count
- detected language (optional)
- complexity score

## PDF
- text layer coverage
- image ratio
- table candidate count
- formula candidate count
- layout variance
- OCR necessity estimate

## DOCX
- heading style availability
- paragraph count
- table count
- nested table indicators
- text box / floating object indicators

## XLSX
- sheet count
- used range density
- merged cell count
- formula ratio
- multi-header indicators
- repeated region indicators

## DOC
- conversion feasibility
- conversion output quality signals
